## [v0.1.0] - 2026-04-19

### Highlights

- Initial release with structured profile schema, JD parsing, and AI-powered resume generation pipeline

### Added

- Structured profile JSON schema for career data (experience, education, skills, achievements)
- Job description parser extracting requirements from markdown files
- Profile-to-JD matcher with keyword-based scoring and gap analysis
- JSON file-based profile storage with default directory `~/.sprofile/profiles/`
- Resume generation service creating tailored resumes from profile + JD
- Cover letter generation service with customization options
- PDF/DOCX export via Pandoc with configurable fonts and margins
- Data migration utilities for converting legacy formats
- `sprofile` CLI with `resume generate` and `cover generate` commands
- Job-match evaluation schema (matcheval) for structured profile-to-JD scoring
- Match category types: technical_skills, leadership, domain_experience, etc.
- Match types: exact, semantic, partial, threshold, boolean, weighted
- AI agent specifications for semantic resume generation pipeline
- `jd-analyst` agent for semantic job description parsing
- `profile-matcher` agent for profile-to-JD matching with scoring
- `resume-generator` agent for tailored resume creation
- `resume-evaluator` agent for LLM-as-Judge quality scoring
- `resume-refiner` agent for feedback-based improvement
- `resume-coordinator` agent for pipeline orchestration
